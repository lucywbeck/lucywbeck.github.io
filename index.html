Froggo

Froggo is a clone of the 1981 arcade game Frogger. I developed a graphical user interface (GUI) that receives user input to support interactive game play. I designed multiple levels that support JSON files, audio, 2D graphics with hitboxes, and scheduled events. I utilized property decorators, generators, and coroutines to create model classes that support 2D animation. I implemented the model-view-controller design pattern to organize code written in Python and Kivy
